# Blueprints

## Customization

Feel free to customize the product and category blueprints. This might be useful if you want to add additional information about a product.

{% hint style="danger" %}
Make sure to **NOT** change any handles of existing fields. If you do, things will blow up.
{% endhint %}

## Assets

The asset container for the product images defaults to `assets`. You may change this to any container you want.

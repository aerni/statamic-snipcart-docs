# Configuration

## Add API key

Add your **Snipcart API key** to your `.env` file. You can find it in your [Snipcart dashboard](https://app.snipcart.com/dashboard/account/credentials).

```bash
SNIPCART_API_KEY=************************
```

## Configure package settings

Make sure to review the package config published to `config/snipcart.php` to get an overview of the available settings and options.

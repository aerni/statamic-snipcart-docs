# Currency

## Code

The code of the currency, e.g. `USD`.

```text
{{ currency:code }}
```

## Name

The name of the currency, e.g. `US Dollar`.

```text
{{ currency:name }}
```

## Symbol

The symbol of the currency, e.g. `$`.

```text
{{ currency:symbol }}
```
